一、确定风格
固定模板

顶部切换「杂志 / 报纸 / 便签」三种风格，每种自带两个预设配色，点一下即可切换。

点顶部「配色」按钮，可以自定义主色 / 辅色 / 点缀色，实时预览效果。

自定义模板

点顶部「＋」按钮，弹窗里有现成的 Prompt，复制给 AI → 把返回的 JSON 保存为 .json 文件 → 上传，顶部会出现新 tab 并自动切换。

可以同时存多个自定义模板；hover tab 出现「×」可单独删除。同样支持配色面板调色。

注意：自定义模板没有持久化，刷新或关闭页面后会消失，把 JSON 文件保存好。

二、排版
导入 Markdown 文件

点编辑区顶部「导入 Markdown 文件」，自动按 # / ## / ### / > / --- 解析成对应块类型。封面和结尾不会被覆盖，需要单独编辑。

直接写：编辑内容

· 添加块：编辑区底部按钮，分正文 / 标题 / 引用 / 分隔线 / 图片等类型，插入到当前编辑块的下方。

· 修改内容：直接点击文字编辑。

· 切换类型：每个块左侧按钮可切换块类型（如 H1 改成 H2）。

· 复制块：块右侧「+」按钮。

· 删除块：块右侧「×」按钮，或多选后按 Delete 键。

· 调整顺序：悬停块左侧 → 出现 ⋮⋮ 手柄 → 拖动到目标位置。

直接写：批量操作

点块右侧「✓」加入多选（变蓝），再点取消；也可以在空白处拖出蓝色框选。

· 点任意选中块的「×」→ 批量删除（或按 Delete 键）

· 点任意选中块的「+」→ 批量复制，插到最后一个选中块下方

· 点 H1/H2 类型按钮 → 批量改层级

· 用 ⋮⋮ 手柄拖动 → 整组移动

直接写：行内格式

选中文字后，点 B / B / B 按钮，或用快捷键 Ctrl+B 加粗、Ctrl+U 下划线、Ctrl+I 斜体。

未选中文字直接点按钮，会对整个块应用格式，再点一次取消。

直接写：高亮重点

选中文字 → 点「高亮」，马克笔效果，颜色跟随点缀色。光标放到高亮文字里 → 点「取消高亮」撤销。

直接写：插入图片

点图片块一次可选多张（最多 9 张），上传完成后显示「✓ 全部已上传」。单张 ≤ 3MB，仅支持 JPG / PNG / WebP / GIF / BMP。

每张图右上角「×」可单独删除；下方「图注」框可加说明，不写就不显示。

预览效果：单图撑满宽度；多图自动居中换行，保持原比例不裁剪。

注意：图片有有效期，复制 HTML 后请尽快粘贴发布；微信发布后会自动转存，之后不受影响。

撤销 / 重做

顶部「↶ 撤销 / ↷ 重做」按钮，或快捷键 Ctrl+Z / Ctrl+Y。有可撤销内容时，撤销按钮会变成黑底提示。

三、发布

① 确认图片块显示「✓ 全部已上传」

② 点右上角「复制 HTML」

③ 打开公众号后台 → 新建图文 → 粘贴 → 预览 → 发布
